USB 和 VGA 接口¶

如果您使用的是旧版MacBook Air或者是Pro，Surface Pro以及各种Windows系统的笔记本电脑，一般配有USB2.0或3.0接口，或直接配有VGA接口。如果具备VGA接口则可以直接将讲台上的VGA电缆与电脑相连，否则需要一个USB转VGA的接口。
